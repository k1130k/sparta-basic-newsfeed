 ### 테스트 커버리지 33% 

 ![image](https://github.com/user-attachments/assets/8ce82593-b7c9-440b-9972-eda958414fa9)

 #####컨트롤러 테스트에서 막혀서 커버리지를 50%를 달성하기 어려움이 있었습니다.

 ### 프로젝트 소감

 ##### 기존 뉴스피드 프로젝트에서 post를 담당을 했었는데 간단했음에도 불구하고 엄청 애를 먹었던 기억이 있습니다.
 ##### 그땐 CRUD를 제대로 몰라서 이해가 안된채로 강의를 참고하면서 작성하다보니 응용이 안됐던게 컸습니다
 ##### 아웃소싱 프로젝트땐 연관관계가 필요했는데 CRUD는 공부를 한번 더 해서 순조롭게 했는데 연관관계에 대한 개념이 없다보니 id값이 null로 되는 문제가 계속발생했었습니다
 ##### 그래서 아웃소싱 프로젝트때도 상당한 아쉬움이 남아서 뉴스피드 복습 프로젝트를 신청했었던건데 팀플로하던걸 나 혼자하고 팀플에서 했던 것보다 더 업그레이드가 되다보니 생각보다 많이걸렸지만
 ##### 팀에서 하던 것을 혼자 다 하다보니 다방면으로 지식을 얻게돼서 아주 좋은 것 같습니다 뉴스피드,아웃소싱 둘다 post만 담당을 헀었는데 이번 과제를 통해 다른 부분도 경험할 수 있어서 도움이 많이 됐습니다
